# sistema-renta-vehiculos
Sistema de renta de vehículos desarrollado como aplicación web, que permite la administración de clientes, vehículos, choferes, rentas y mantenimientos. Incluye funcionalidades de CRUD completo, control de disponibilidad de vehículos, autenticación básica de usuario y módulo de mantenimiento para el control del estado de los vehículos.
