# 🚗 Sistema de Renta de Vehículos

## 📌 Descripción
Sistema web para la gestión de renta de vehículos, que permite administrar clientes, vehículos, choferes, rentas y mantenimientos. Incluye autenticación básica de usuarios y control del estado de los vehículos.

---

## ⚙️ Tecnologías utilizadas
- Node.js
- Express
- MongoDB
- HTML
- CSS
- JavaScript

---

## 📁 Estructura del proyecto
- config/
- controllers/
- models/
- routes/
- public/
- server.js

---

## 🚀 Instalación

Clonar el repositorio:
git clone URL_DEL_REPOSITORIO

Entrar a la carpeta:
cd nombre-del-proyecto

Instalar dependencias:
npm install

---

## ▶️ Ejecución

node server.js

Abrir en navegador:
http://localhost:3000

## 📊 Módulos del sistema

- Clientes (CRUD)
- Vehículos (CRUD)
- Choferes (CRUD)
- Rentas (CRUD)

- Login y cierre de sesión
- Control de disponibilidad de vehículos

---

## 📌 Funcionalidades principales

- Registro, edición y eliminación de datos
- Asignación de vehículos en renta
- Cambio de estado (Disponible / Rentado)
- Control de mantenimiento
- Interfaz web con consumo de API
